# Story 1.5: Containerize the Application with Docker

**Status:** Ready for Review

**Story:**
As a developer, I want to containerize the application using Docker, so that I can create a portable and reproducible environment for development and deployment.

**Acceptance Criteria:**
1.  A `Dockerfile` is created in the project root.
2.  The Docker image builds successfully.
3.  The application runs in a Docker container.
4.  The application is accessible from the host machine.

**Dev Notes:**

*   Use an official OpenJDK 11 image as the base image.
*   The `build.gradle` file should be used to build the application inside the Docker image.
*   The application should be packaged as a WAR file and deployed to a Tomcat server inside the container.
*   The container should expose the application's port (e.g., 8080) to the host machine.

**Tasks / Subtasks:**

1.  [x] (AC: 1) Create a `Dockerfile` in the project root.
2.  [x] (AC: 2) Build the Docker image using the `docker build` command.
3.  [x] (AC: 3, 4) Run the Docker container and verify that the application is running and accessible.

**Project Structure Notes:**
A new file `Dockerfile` will be created in the project root.
